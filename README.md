# React Dashboard CRUD (TypeScript)

## Goal

Build a scalable CRUD dashboard using React + TypeScript with clean architecture and reusable components.

---

## Features

1 - CRUD Operations

- Create, Read, Update, Delete functionality
- Connected with json-server as a fake API
- Dynamic data rendering in table/dashboard view

---

2 - Pagination

- Server-side pagination using `_page` and `_limit`
- Reading `x-total-count` from headers
- Calculating total pages dynamically
- Reusable Pagination Component

---

3 - Dark / Light Mode

- Toggle between dark and light themes
- Persist theme using localStorage
- Global styling applied using root class

---

4 - Reusable Components

- Input Component
- Button Component
- Form Components
- Pagination Component

- Designed to be flexible and reusable across the app

---

5 - Forms & Validation

- Controlled form inputs
- Form error handling
- Dynamic error messages

---

6 - TypeScript Integration

- Strong typing for data models
- Interfaces for API responses
- Improved code maintainability and scalability

---

7 - JSON Server (Mock API)

- Simulates backend using db.json
- Supports CRUD operations
- Easy setup for local development

---

8 - Responsive Design

- Fully responsive dashboard layout
- Works across different screen sizes
